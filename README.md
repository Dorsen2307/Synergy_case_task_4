## 📝 Описание проекта

Интерактивное веб-приложение - калькулятор с современным дизайном и анимациями. Проект разработан в учебных целях для демонстрации работы с JavaScript, CSS и HTML.

## ✨ Функциональность

- ✅ Ввод двух чисел
- ✅ Четыре математические операции:
  - ➕ Сумма
  - ➖ Разность
  - ✖️ Произведение
  - ➗ Деление
- ✅ Автоматическая очистка полей после вычисления
- ✅ Визуальное отображение результатов
- ✅ Обработка ошибок (деление на ноль, пустые поля, ввод не числа)
- ✅ Адаптивный дизайн

## 🎨 Особенности интерфейса

- Градиентный фон
- Плавные анимации при загрузке
- Эффект наведения на кнопках
- Цветовая индикация результатов:
  - 🟢 Зеленый градиент - успешное вычисление
  - 🔴 Красный градиент - ошибка
- Анимация встряхивания при ошибке

## 🛠 Технологии

- **HTML5** - структура приложения
- **CSS3** - стилизация и анимации
- **JavaScript** - логика работы калькулятора
